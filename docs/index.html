openshiftClient

Builds the rest client based on provided or default kubernetes configuration.

If you want to use a username/password combo to authenticate to Openshift, you might do something like the example.

openshiftClient(settings: Object): Promise
Parameters
settings (Object = {}) kubernetes configuration
Returns
Promise: Returns the Rest Client
Example
settings.config = {
  url: process.env.CLUSTER_URL,
    auth: {
      username: process.env.ADMIN_USERNAME,
      password: process.env.ADMIN_PASSWORD
    },
    insecureSkipTlsVerify: true
};
